# RenThing - Modern Rental Marketplace

RenThing is a revolutionary rental marketplace that connects people who have items to rent with those who need them. Our platform makes it easy to list, discover, and rent anything from electronics to vehicles to tools and more.

## Features

- **Modern UI/UX Design**: Clean, intuitive interface with a focus on user experience
- **Comprehensive Search**: Find rentals by category, location, and price range
- **Secure Booking System**: Safe and reliable booking with integrated payments
- **User Profiles**: Build trust with ratings and verification systems
- **Messaging System**: Communicate directly with renters and owners
- **Mobile Responsive**: Works seamlessly on all devices

## Technology Stack

- **Frontend**: React.js with Material-UI
- **Backend**: Node.js with Express
- **Database**: SQLite
- **Authentication**: JWT-based secure authentication
- **Payments**: Stripe integration
- **Maps**: Google Maps API

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Visit `http://localhost:3000` in your browser
